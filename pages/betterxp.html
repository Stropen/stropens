Terminalden Çalıştırma

Ön Bilgi: BetterXP Fedora, Debian, Solus tabanlı GNU/Linux dağıtımlarına destek verir. Windows işletim sistemleri bu hizmetin dışında kalmaktadır.

1.Git ile Python3 paketini yüklemek gerekir.

Peki bu tür dağıtımlara nasıl yükleriz?

Fedora: Bu tabanı kullanan bir dağıtımda terminale sudo apt install git python3 -y yazmalısınız.

Debian: Bu tabanı kullanan bir dağıtımda terminale sudo dnf install git python3 -y yazmalısınız.

Solus: Bu tabanı kullanan bir dağıtımda terminale sudo eopkg it git python3 -y yazmalısınız.

Artık BetterXP için terminale git clone https://github.com/androidprotmmas/BetterXP-Installer.git ; cd BetterXP-Installer ; sudo python3 installer.py yazarak keyfinize bakabilirsiniz, araç kiti kurulacaktır.
