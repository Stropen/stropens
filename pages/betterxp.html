Terminalden Çalıştırmak

Ön Bilgi: BetterXP Fedora, Debian, Solus tabanlı GNU/Linux dağıtımlarına destek verir. Windows işletim sistemleri bu hizmetin dışında kalmaktadır.

Git ile Python3 paketini yüklemek gerekir.

Peki belirtilen dağıtımlara nasıl yükleriz?

Fedora: Bu tabanı kullanan bir dağıtımda terminale sudo apt install git python3 -y yazmalısınız.

Debian: Bu tabanı kullanan bir dağıtımda terminale sudo dnf install git python3 -y yazmalısınız.

Solus: Bu tabanı kullanan bir dağıtımda terminale sudo eopkg it git python3 -y yazmalısınız.



Artık BetterXP için terminale aşağıda yazan komutu yazdığınızda araç kiti çalışacaktır. İyi günler dileriz.

git clone https://github.com/androidprotmmas/BetterXP-Installer.git ; cd BetterXP-Installer ; sudo python3 installer.py"

El İle Kurmak

1.Ana sayfa üzerinden zip dosyasını indirin.

2.Zip dosyasını çıkartın.

3.Python 3.9.6 sürümünü yükleyin.

4.İnstaller.py dosyası ile kurulumu yapın.

5.BetterXP.desktop dosyasını çalıştırın. İyi günler dileriz.

BetterXP Özellikleri

-Masaüstü ortamı kurar ve kaldırır.
